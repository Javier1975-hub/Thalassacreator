# Thalassacreator 🚢

Aplicación web para crear flotas de barcos antiguos.  
Permite:
- Seleccionar el tipo de barco, nacionalidad, maestría y opciones.
- Añadir múltiples barcos a la flota.
- Calcular los puntos totales de la flota.
- Exportar un archivo PDF con la lista de barcos y los puntos.

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Usa los botones para crear tu flota.
3. Haz clic en "Descargar PDF" para exportar la flota.

## Estructura
- `index.html`: página principal.
- `css/style.css`: estilos.
- `js/main.js`: lógica principal.
- `assets/`: carpeta para imágenes o íconos.

¡A navegar los mares! ⚓
